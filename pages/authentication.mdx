---
title: "Authentication"
description: "Access to the Award Force API is secured via API Keys. You may generate as many keys as you like yet each key belongs to the one account. The scope of the key determines the permissions of the key:"
---

Read only - for read endpoints only  
Read / Write - for read and write endpoints

Multiple email addresses may be specified for receiving notification emails.

## Generate API key
<Steps>
<Step title="In Award Force navigate to Settings > Integrations > API Keys"/>
<Step title="Click Generate API Key"/>
<Step title="Enter Key name / purpose"/>
<Step title="Select Scope">
  
  a. Read only - for read endpoints only

  b. Read / Write - for read and write endpoints
</Step>
<Step title="Specify notification email addresses (optional). Separate each address with a comma (,) or a semicolon (;)"/>
<Step title="Click Save"/>
</Steps>


## Edit API key
<Steps>
<Step title="In Award Force navigate to Settings > Integrations > API Keys"/>
<Step title="Click on the Key name / purpose of the key you wish to edit OR click on the overflow action menu of the key you wish to edit and click Edit"/>
<Step title="Update Key name / purpose"/>
<Step title="Update Scope">
  
  a. Read only - for read endpoints only

  b. Read / Write - for read and write endpoints
</Step>
<Step title="Update notification email addresses (optional). Separate each address with a comma (,) or a semicolon (;)"/>
<Step title="Click Save"/>
</Steps>
## Revoke API key
<Steps>
<Step title="In Award Force navigate to Settings > Integrations > API Keys"/>
<Step title="Click on the overflow action menu of the key you wish to edit and click Revoke"/>
<Step title="Click OK in the confirm dialog"/>
</Steps>
