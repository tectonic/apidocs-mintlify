## Welcome to the Award Force API

The Award Force API enables your engineering team or developers to programmatically integrate with the Award Force platform, allowing you to pass data between Award Force and other websites or applications.

## Available Resources

With our API, you can **list**, **get**, **create**, **update**, **delete**, and **upload** data across many different resources:

<AccordionGroup>
  <Accordion title="Core Resources">
    - Account
    - Season
    - User
    - Webhook
  </Accordion>

  <Accordion title="Entry Management">
    - Entry
    - Entry (realtime)
    - Attachment
    - Field
    - File
    - Contributor
  </Accordion>

  <Accordion title="Judging & Evaluation">
    - Assignment
    - Review task
    - Round
    - Score set
  </Accordion>

  <Accordion title="Organization">
    - Category
    - Chapter
    - Leaderboard
  </Accordion>

  <Accordion title="Commerce">
    - Order
    - Tax
  </Accordion>
</AccordionGroup>
